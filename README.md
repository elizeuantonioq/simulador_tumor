# Simulador de Crescimento de Tumor 🧬

Este projeto consiste num simulador desenvolvido em Python que utiliza modelos matemáticos para prever o crescimento de células tumorais ao longo do tempo. Trabalho desenvolvido para o **Checkpoint 2** do curso de Engenharia de Software da FIAP.

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**: Linguagem principal.
- **SymPy**: Biblioteca de matemática simbólica utilizada para definir e resolver as equações de crescimento de forma precisa.

## 🚀 Como Executar o Projeto

### 1. Pré-requisitos
Como o projeto utiliza a biblioteca `sympy`, precisa instalá-la antes de colocar o código. Abra o seu terminal e digite:

```bash
pip install sympy
```

### 2. Execução
Após a instalação, basta colocar o ficheiro principal:

```bash
python simulador.py
```

## 📋 Funcionalidades
- **Definição simbólica**: Equações de crescimento (Exponencial/Gompertz) tratadas matematicamente.
- **Cálculo dinâmico**: Previsão do volume do tumor com base no tempo e na taxa de crescimento.
- **Estrutura preparada**: Código pronto para evolução com cálculos de derivadas (velocidade de crescimento).
